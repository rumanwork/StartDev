Установка Xcode

При разработке приложения, вы используете Xcode - интегрированную среду разработки Apple (IDE). Xcode предоставляет все необходимое для создания больших Mac приложений. Она включает в себя редактор исходного кода, графический редактор пользовательского интерфейса, и многие другие функции. Xcode состоит из одного окна, называемого рабочей областью, которое представляет большинство инструментов, необходимых для разработки приложений.

Важно. Эта интсрукция носит предварительный характер, она предназначена для работы с последним программным обеспечением. Для выполнения руководства скачайте последний Xcode и обновитесь до
OS X v10.8.

Для начала:

  1. Скачайте поледнюю версию Xcode.

    Открытие Mac App Store на вашем Mac, найдите Xcode и нажмите Скачать. Mac App Store устанавливается вместе с Mac OS X версии 10.7 и выше. Если у вас более ранняя версия Mac OS X, вам нужно обновиться.

  2. Зарегистрироваться в программе разработчиков Mac.

    После регистрации в программе, у вас появится доступ к инструментам и ресурсам, необходимым для распространения вашего приложения. Вы узнаете больше об этих средствах позже.
